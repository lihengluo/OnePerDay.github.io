## 正则表达式简述

#### 语法

```txt
匹配任意一个字符
[0-9]
[a-z]
[A-Z]
[0-9a-z]
-[0-9]
.
贪婪非贪婪
.*? / .* 
| 
() 保存内容
\d \D
\w \W
{} 次数
^ 
$
```

#### python

```python
import re
pattern = ''
str = ''
re.search(pattern, str)
re.match(pattern, str, flag = re.I|re.S|re.W)
# .group() / .groups()
re.findall(pattern, str)
```
